Device Usage Analysis

The device usage patterns reveal important insights about user preferences across different services:

  • MMT: Desktop dominates (62.5%), suggesting users prefer larger screens for this service.
  • YATRA: Desktop leads (48.6%), but with a more balanced distribution across devices.
  • GOIBIBO: iOS (37.6%) is most popular, indicating a strong mobile focus among this service's users.

These differences may reflect variations in user interface quality, target demographics, or marketing approaches across the services.

Business Recommendations
  • Optimize the MMT experience specifically for desktop users, focusing on detailed comparison features.
  • Enhance the GOIBIBO mobile experience, particularly for iOS users.
  • Consider device-specific promotions (e.g., mobile-only discounts for GOIBIBO).
  • Identify opportunities to improve underperforming platforms (e.g., MMT's mobile experience).
  • Analyze the relationship between device usage and booking value to identify high-value customer segments.
